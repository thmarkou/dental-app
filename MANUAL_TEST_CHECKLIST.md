## 1. Κλινική & ασθενείς

### Ασθενής
- [ ] Νέος ασθενής με ΑΦΜ 9 ψηφία (για myDATA προσομοίωση)
- [ ] Επεξεργασία / φωτογραφία / GDPR συγκατάθεση

### Οδοντόγραμμα
- [ ] Καταχώρηση θεραπείας **ανά δόντι** → ενημέρωση chart
- [ ] Καταχώρηση **γενικής** θεραπείας (χωρίς δόντι)
- [ ] Αν υπάρχει ανοιχτό plan item: alert «σχέδιο» → **Ολοκλήρωση + χρέωση**

### Σχέδιο θεραπείας
- [ ] Νέο σχέδιο, φάσεις, item με **πολλά δόντια**
- [ ] **Ολοκλήρωση + χρέωση** → **N χρεώσεις** στο λογιστήριο (όχι μία)
- [ ] Odontogram: planned overlay, χωρίς διπλή χρέωση
- [ ] PDF σχεδίου (κοινοποίηση)
- [ ] Εναλλακτικές προτάσεις (v16)

### Λογιστήριο ασθενή
- [ ] Χρέωση + πληρωμή → σωστό υπόλοιπο
- [ ] Έκδοση απόδειξης από πληρωμή (ledger)
- [ ] myDATA προσομοίωση (με έγκυρο ΑΦΜ)

## 4. Αποθήκη (v19 + BOM)

### Βασική αποθήκη
- [ ] **Αναφορές** → Αποθήκη: KPIs (αξία, κατανάλωση μήνα), πρόσφατες κινήσεις
- [ ] Νέο είδος, κίνηση Πρόσθεσε/Αφαίρεσε
- [ ] Θεραπεία στο chart → prompt αφαίρεσης υλικών (αν έχει BOM)
- [ ] Φίλτρο «Λίγα τεμάχια»

### Θεραπείες ↔ υλικά (462ef6a)
- [ ] Αποθήκη → **Σύνδεση καταλόγου θεραπειών με υλικά**
- [ ] Ορισμός BOM (π.χ. Στεφάνη → 1× υλικό)
- [ ] Νέα θεραπεία στο chart → alert **Αφαίρεση από αποθήκη;** → απόθεμα μειώνεται
- [ ] Σχέδιο → Ολοκλήρωση + χρέωση (πολλά δόντια) → ποσότητα × N δόντια στο alert

### Αποθήκη advanced (B3+ / Φάση 3)
- [ ] **Ρυθμίσεις** → «Αποθήκη & υλικά» → **Αυτόματη αφαίρεση υλικών** ON
- [ ] Θεραπεία με BOM → **χωρίς** dialog confirm (αν stock OK)· OFF → dialog όπως πριν
- [ ] Αποθήκη → επεξεργασία είδους → **Ημερομηνία λήξης** (YYYY-MM-DD)
- [ ] Φίλτρα: **Λήγουν σύντομα** (≤30 ημέρες) · **Ληγμένα**
- [ ] Badges «Λήγει σύντομα» / «Ληγμένο» στη λίστα
